# 🍽️ HS Food Point

A clean, modern, and fully responsive restaurant website built using **HTML5** and **CSS3**. This project showcases a complete single/multi-section website for a food business — including a hero landing page, about section, interactive food menu, and a contact form.



## 📸 Preview

| Home | About |
|------|-------|
| Hero section with call-to-action | Welcome section with description |

| Menu | Contact |
|------|---------|
| Food items with pricing | Contact form + business details |

## ✨ Features

- 🎨 **Modern Hero Section** — Eye-catching gradient banner with brand name, tagline, and CTA button
- 📖 **About Section** — Brief intro about the restaurant with a clean two-column layout
- 🍕 **Interactive Food Menu** — 10+ food items displayed as cards with emojis/icons and prices
- 📞 **Contact Section** — Address, phone, email, and a working contact form UI (Name, Email, Phone, Message)
- 📱 **Fully Responsive** — Adapts smoothly across desktop, tablet, and mobile screens
- 🎯 **Simple Navigation** — Sticky navbar with Home, Menu, About, and Contact links

## 🛠️ Built With

- **HTML5** — Semantic structure
- **CSS3** — Flexbox / Grid, gradients, hover effects, responsive design

## 📌 Sections Breakdown

1. **Home** – Welcome banner with restaurant name, tagline (*Fresh • Tasty • Homestyle*), and an "Order" button
2. **About** – Short brand story explaining the restaurant's philosophy
3. **Menu** – Grid layout displaying dishes like Cheese Pizza, Veg Burger, Pasta, Noodles, Biryani, and more with prices
4. **Contact** – Business address, phone, email, and a contact form for inquiries/table bookings

## 🎯 What I Learned

- Structuring a multi-section website using semantic HTML
- Building responsive layouts using Flexbox/Grid
- Designing clean UI cards and hover interactions
- Creating a consistent color theme and typography across sections
